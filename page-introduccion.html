Introducción al sistema operativo

¿Qué es linux y por que todo el mundo habla de el?

Linux no es solo un sistema operativo; es el motor que mueve internet, supercomputadoras y casi todos los servidores del planeta. A diferencia de Windows o MacOs, Linux es de "Código abierto", lo que significa que cualquiera puede verlo, modificarlo y mejorarlo.

El kernel

El corazon del sistema que gestiona el hardware.

Distribucion (Distro)

El kernel + Herramientas + Entorno Grafico (Ej.: Debian, Ubuntu).
