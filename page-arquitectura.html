Arquitectura del Sistema

Las capas que separan tus programas del hardware real.

Linux organiza el software en dos mundos principales para garantizar la seguridad y estabilidad.

  • User Space:Donde viven tus aplicaciones (Navegador, Terminal, Juegos). Tienen acceso limitado por seguridad.
  • Kernel Space:Donde vive el núcleo. Tiene controltotal sobre la CPU, la RAM y los discos.
