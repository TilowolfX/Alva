Arquitectura Del Sistema

Las capas que seperan tus programas del hardware real.

Linux organiza el software en dos mundos principales para garantizar la seguridad y estabilidad.

User Space: Donde viven tus aplicaciones (Navegador, Terminal,Juegos,Wps office). Tienen acceso limitado por seguridad

Kernel Space: Donde vive el nucleo. Tiene el control total sobre la CPU, la Ram y los discos.
